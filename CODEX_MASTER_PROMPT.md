====================================================================
ОБЩАЯ АРХИТЕКТУРА ПЛАТФОРМЫ
====================================================================

Разрабатывается локальная веб-платформа (core platform),
работающая в локальной сети (LAN).

Платформа:
- является ЯДРОМ
- является КОНСТРУКТОРОМ
- НЕ содержит бизнес-логики
- НЕ знает предметную область модулей

ВСЯ бизнес-логика реализуется ТОЛЬКО
через подключаемые модули в будущем.
